## INTRODUCTION
This project presents an interactive Microsoft Excel dashboard built using the Olist Brazilian E-commerce dataset. The dashboard analyses over 99,000 customer orders placed between 2016 and 2018 to evaluate business performance, customer satisfaction, delivery efficiency, and payment behaviour.

The project transforms a relational dataset into an interactive dashboard that enables users to explore key business metrics through dynamic filters and visualisations.

## Objective
The objective of this project was to demonstrate an end-to-end Excel data analytics workflow by transforming raw relational e-commerce data into meaningful business insights. The dashboard was designed to help stakeholders monitor sales performance, evaluate delivery operations, understand customer behaviour, and identify opportunities for operational improvement.

## Excel Skills Used
- 🔄 **Power Query** – data cleaning and transformation
- 🧩 **Power Pivot** – relational data modelling
- 🧮 **DAX Measures** – KPI and business metric calculations
- 📊 **PivotCharts** – interactive data visualisation
- 📋 **PivotTables** – data aggregation and analysis
